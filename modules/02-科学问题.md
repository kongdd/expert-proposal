# 科学问题凝练与撰写技能 (Scientific Question Writing Skill)

## 概述
科学问题是项目申请书的"灵魂"，是区分普通申请与优秀申请的关键。本技能专注于指导AI理解什么是真正的科学问题、如何从研究背景中凝练科学问题、以及如何规范表述科学问题。

---

## 一、科学问题的本质理解

### 1. 什么是科学问题

科学问题 ≠ 技术问题 ≠ 工程问题 ≠ 研究内容

| 类型 | 示例 | 特征 |
|------|------|------|
| ❌ 技术问题 | "如何提高模型精度" | 方法导向，关注"怎么做" |
| ❌ 工程问题 | "如何建设监测系统" | 应用导向，关注"建什么" |
| ❌ 研究内容 | "分析XX的时空变化特征" | 任务导向，关注"做什么" |
| ✅ 科学问题 | "XX时空分异的驱动机制是什么" | 认知导向，关注"是什么/为什么" |

### 2. 科学问题的核心特征

```
┌─────────────────────────────────────────────────────────┐
│                    科学问题五要素                        │
├─────────────────────────────────────────────────────────┤
│  ① 科学性：涉及自然规律、机理、机制的认知               │
│  ② 前沿性：当前学界尚未充分回答或存在争议               │
│  ③ 明确性：问题边界清晰，可被理解和讨论                 │
│  ④ 可验证性：可通过科学方法获得答案                     │
│  ⑤ 价值性：解决后能推动理论发展或实践应用               │
└─────────────────────────────────────────────────────────┘
```

### 3. 科学问题的层次结构

```
第一层：根本性科学问题（宏观）
    │
    ├── 第二层：核心科学问题（本项目聚焦）
    │       │
    │       ├── 第三层：具体科学问题①
    │       ├── 第三层：具体科学问题②
    │       └── 第三层：具体科学问题③
    │
    └── （其他分支，非本项目关注）
```

**示例**：
- **根本问题**：全球变暖如何影响陆地生态系统碳循环？
  - **核心问题**：气候变暖背景下高寒草地碳源/汇功能如何转变？
    - **具体问题①**：增温如何改变植被-土壤碳交换过程？
    - **具体问题②**：冻土退化对土壤有机碳分解的影响机制？
    - **具体问题③**：水热耦合变化如何调控生态系统碳通量？

## 二、科学问题的类型学

### 1.  按认知层次分类

| 类型 | 核心追问 | 典型表述 | 示例 |
|------|----------|----------|------|
| **描述型** | 是什么/有什么 | "... 的特征/格局/模式是什么" | "青藏高原湖泊扩张的时空格局是什么" |
| **解释型** | 为什么/怎么发生 | "... 的形成机理/驱动机制是什么" | "高原湖泊扩张的水文驱动机制是什么" |
| **关系型** | 如何相互作用 | "...与...的耦合关系/交互作用是什么" | "气候变化与冰川消融对湖泊扩张的协同效应" |
| **预测型** | 将会怎样 | "...的演变趋势/未来情景是什么" | "未来气候情景下高原湖泊的演变趋势" |

### 2. 按学科特征分类

| 学科领域 | 常见问题模式 | 关键词 |
|----------|--------------|--------|
| 生命科学 | 分子机制、信号通路、调控网络 | 调控、表达、通路、靶点 |
| 地球科学 | 过程机理、时空演变、耦合关系 | 过程、演化、响应、反馈 |
| 物理学 | 物理机制、相变规律、动力学特性 | 机制、相变、动力学、量子效应 |
| 工程科学 | 失效机理、性能演化、优化原理 | 机理、失效、演化、耦合 |
| 社会科学 | 影响因素、作用机制、演化规律 | 机制、路径、效应、治理 |

## 三、科学问题的凝练方法

### 1. 凝练路径：从现象到本质

```
         观察现象
            ↓
    ┌───────────────┐
    │   追问"为什么"  │ ← 核心思维动作
    └───────────────┘
            ↓
         发现矛盾
     （理论与现实不符/
      现有认知无法解释）
            ↓
    ┌───────────────┐
    │   聚焦知识缺口  │ ← 文献综述支撑
    └───────────────┘
            ↓
       提炼科学问题
```

### 2.  四步凝练法

#### 第一步：现象描述
- 准确描述观察到的现象或发现的矛盾
- **示例**：近年来观测发现，北极海冰快速消融，但不同区域消融速率差异显著。

#### 第二步：追问本质
- 连续追问3-5个"为什么"
- **示例**：
  - 为什么不同区域消融速率不同？→ 可能与海洋热通量有关
  - 为什么海洋热通量存在差异？→ 可能受洋流和大气环流影响
  - 这些因素如何耦合作用？→ **知识缺口出现**

#### 第三步：文献验证
- 检索现有研究，确认问题的前沿性
- 明确"已知"与"未知"的边界
- **关键句式**："现有研究表明... ，但... 尚不清楚"

#### 第四步：问题表述
- 用规范语言凝练科学问题
- 确保问题可被本项目回答

### 3. 常见凝练误区

| 误区 | 错误示例 | 正确示例 |
|------|----------|----------|
| **过于宽泛** | "气候变化对生态系统的影响" | "增温如何改变高寒草甸的碳源/汇功能" |
| **过于狭窄** | "某地区2020年降水异常的原因" | "东亚夏季风年际变异的动力学机制" |
| **技术化** | "如何提高遥感反演精度" | "复杂地表条件下辐射传输的尺度效应" |
| **模糊不清** | "研究XX的相关问题" | "XX与YY的时空耦合关系及驱动机制" |
| **缺乏新意** | 重复已被充分回答的问题 | 在已有认知基础上的新突破点 |

## 四、科学问题的表述规范

### 1. 表述结构模板

#### 模板A：要素完整型
```
在...背景/条件下，...的... 机理/机制/规律是什么？
```
**示例**：
> 在全球变暖背景下，青藏高原冻土退化对土壤碳释放的调控机制是什么？

#### 模板B：关系阐释型
```
... 与...之间的耦合关系/交互作用/反馈机制是什么？
```
**示例**：
> 植被恢复与土壤微生物群落之间的协同演化机制是什么？

#### 模板C：过程解析型
```
... 过程中，...如何影响/调控/驱动...？
```
**示例**：
> 极端干旱过程中，植物根系如何调控土壤水分再分配？

#### 模板D：陈述句型
```
揭示/阐明... 的...机理/规律，明确...的关键因素/调控途径
```
**示例**：
> 揭示深层土壤有机碳分解的微生物驱动机制，阐明温度敏感性的调控因素。

### 2. 关键动词选择

| 认知层次 | 推荐动词 | 避免动词 |
|----------|----------|----------|
| 揭示机理 | 揭示、阐明、解析 | 研究、分析、探讨 |
| 发现规律 | 发现、识别、确定 | 了解、调查、观察 |
| 建立关系 | 建立、构建、厘清 | 看看、考察、涉及 |
| 提出理论 | 提出、发展、创建 | 总结、归纳、整理 |

### 3. 修饰语使用

**增强科学性的修饰语**：
- 限定范围：在... 条件下、针对...类型、基于... 尺度
- 强调深度：从... 层面、在...机制水平、从...角度
- 体现系统性：系统揭示、全面阐明、定量解析

**避免使用的表述**：
- 过于绝对："首次发现"（需有充分依据）
- 模糊不清："相关问题"、"若干方面"
- 口语化："搞清楚"、"弄明白"

## 五、科学问题的组织架构

### 1. 数量要求
- **基金项目**：一般2-3个科学问题为宜
- 过少：深度不够或覆盖面窄
- 过多：焦点分散，难以深入

### 2. 问题间逻辑关系

#### 类型一：递进关系
```
科学问题① ──→ 科学问题② ──→ 科学问题③
  (基础)        (深化)         (拓展)
```
**示例**：
1. XX的时空分布格局是什么？（描述）
2.  XX时空分异的驱动机制是什么？（解释）
3. 未来情景下XX将如何演变？（预测）

#### 类型二：并列关系
```
        ┌── 科学问题① (维度A)
核心主题 ├── 科学问题② (维度B)
        └── 科学问题③ (维度C)
```
**示例**：
1. 增温对植物生理过程的影响机制（生物维度）
2.  增温对土壤碳氮循环的影响机制（土壤维度）
3. 植物-土壤相互作用的响应与反馈（耦合维度）

#### 类型三：层次关系
```
科学问题① (宏观尺度)
    │
    ├── 科学问题② (中观过程)
    │
    └── 科学问题③ (微观机制)
```

### 3. 与研究内容的对应

```
┌─────────────────────────────────────────────────────────┐
│  科学问题①  ←──对应──→  研究内容1、研究内容2           │
│  科学问题②  ←──对应──→  研究内容3                      │
│  科学问题③  ←──对应──→  研究内容4、研究内容5           │
└─────────────────────────────────────────────────────────┘
```

**衔接表述**：
> "为解决上述科学问题，本项目设置以下研究内容：针对科学问题①，通过研究内容1和2，从... 角度揭示...机理；针对科学问题②，..."

## 六、科学问题写作范例

### 范例一：生态学方向

**研究背景关键信息**：
- 全球变暖导致高寒草地碳循环发生变化
- 现有研究多关注地上部分，土壤碳动态研究不足
- 冻土碳库巨大，其稳定性受温度影响尚不明确

**凝练的科学问题**：

> 围绕"气候变暖背景下高寒草地土壤碳稳定性"这一核心问题，本项目拟解决以下三个关键科学问题：
>
> **（1）增温如何改变高寒草地土壤有机碳的组分特征与分布格局？**
> 
> 聚焦土壤有机碳的化学组成与空间分异，揭示增温对不同活性碳组分（活性碳、慢性碳、惰性碳）的差异性影响，明确土壤碳库脆弱性的空间异质性。
>
> **（2）冻土退化过程中土壤微生物群落如何响应并驱动碳分解？**
>
> 从微生物生态学角度，解析冻土融化对土壤微生物群落组成、功能基因及酶活性的影响，阐明"冻土退化-微生物演替-碳分解"的耦合机制。
>
> **（3）植被变化与土壤碳动态之间的反馈机制是什么？**
>
> 探讨植被物候变化、生产力改变对土壤碳输入的影响，揭示植被-土壤系统的碳循环反馈效应及其对生态系统碳源/汇功能的调控作用。

### 范例二：信息科学方向

**研究背景关键信息**：
- 深度学习在复杂场景识别中广泛应用
- 现有模型在分布外样本上泛化能力差
- 因果推理被认为是提升泛化能力的关键

**凝练的科学问题**：

> 本项目聚焦"深度学习模型的因果泛化机制"这一前沿问题，拟解决以下关键科学问题：
>
> **（1）复杂视觉场景中因果特征与相关性特征的识别与分离机制是什么？**
>
> 探索从高维视觉数据中自动识别因果不变特征的理论方法，建立因果特征与伪相关特征的解耦框架。
>
> **（2）因果约束如何提升深度神经网络的分布外泛化能力？**
>
> 揭示因果不变性原理与深度表示学习的内在联系，阐明因果正则化对模型泛化边界的理论影响。

## 七、质量评估与修订

### 自检清单

**科学性检验**：
- [ ] 问题是否指向自然规律/机理/机制？
- [ ] 是否超越了方法/技术层面？
- [ ] 是否具有理论深度？

**前沿性检验**：
- [ ] 现有文献是否已充分回答该问题？
- [ ] 问题是否反映领域发展趋势？
- [ ] 是否存在明确的知识缺口？

**明确性检验**：
- [ ] 问题边界是否清晰？
- [ ] 他人能否准确理解问题内涵？
- [ ] 是否避免了模糊表述？

**可行性检验**：
- [ ] 本项目是否有能力回答该问题？
- [ ] 研究周期内能否取得实质进展？
- [ ] 研究方法是否足以支撑问题解答？

**关联性检验**：
- [ ] 科学问题与研究内容是否对应？
- [ ] 问题之间逻辑关系是否清晰？
- [ ] 是否回应了立项依据中的知识缺口？

### 常见修订策略

| 问题 | 修订策略 |
|------|----------|
| 过于宽泛 | 添加限定条件（时空范围、研究对象、特定条件） |
| 过于技术化 | 追问"为什么需要这项技术"，回归科学本质 |
| 表述模糊 | 使用具体的科学术语，明确核心概念 |
| 新意不足 | 突出与已有研究的差异，强调新视角/新发现 |
| 逻辑松散 | 重新梳理问题间关系，建立清晰的逻辑链条 |

## 八、AI写作辅助提示词模板

当需要帮助用户凝练科学问题时，可使用以下思考框架：

```
【输入信息】
- 研究领域：
- 研究对象：
- 观察到的现象/矛盾：
- 现有研究的不足：

【凝练过程】
1. 核心追问：这个现象背后的"为什么"是什么？
2. 知识缺口：现有理论/认知无法解释什么？
3.  本质问题：需要揭示什么机理/规律/关系？
4. 问题层次：该问题属于描述型/解释型/关系型/预测型？

【输出格式】
科学问题表述：
- 使用规范的学术表达
- 体现科学性和前沿性
- 明确问题边界和核心内涵
```
